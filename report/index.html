E-Commerce Multi-Vendor Platform

Mini Project Report & Submission Dossier
Abstract Overview: This mini-project details the end-to-end design, implementation, and testing of a complete multi-vendor marketplace platform. Built using Django REST Framework (DRF) for the stateless backend API and React.js for a responsive Single-Page Application (SPA), the system demonstrates role-based access control (RBAC), database transaction consistency, secure JWT authentication with refresh token rotation, and modular frontend state management.
